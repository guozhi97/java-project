三峡大学计算机协会成立于2001年,她是一群热爱计算机的99级校友一手创办的。现如今,在计算机与信息学院团委的支持下,由学生自主管理。 协会本着为广大学生普及计算机知识，提高同学们的计算机水平为宗旨。经历了十七年的风风雨雨，协会在历届各位协会领导人的努力下，逐渐发展壮大，并在探索中取得了辉煌佳绩。

  • 协会全称三峡大学计算机协会
  • 成立时间2001年
  • 挂靠院系三峡大学计算机与信息学院
  • 现任会长郑枫

协会内部结构

三峡大学计算机协会成立于2001，经过十几代计协人的努力，计协不断发展壮大，协会内部组织机构也越发完善，2014年新增编程部，2015年科技部被兴趣小组取代，兴趣小组由硬件组、建站组、办公组与美工组这四个部门组成，同年计协成立的创新创业部。2017年，宣传部和美工组合并改为美宣部，建站组和硬件组合并为技术部，减去了外联部。2018年，再次建立建站组。

协会内部结构依次为会长团、部长团、干事群体

社团荣誉

协会多次荣获“三峡大学校级优秀社团”，“三峡大学社团之星”等荣誉。

协会于2009年在湖北省楚天电子文化活动中获得“楚天电子文化节优秀组织”荣誉称号。2013年11月份，协会承办了“2013-2014年度微软创新之旅全国巡讲（宜昌站）”活动，作为微软全国巡讲在宜昌的唯一一站，活动取得了圆满成功，并获得微软特别嘉奖。

2013-2014年获三峡大学校优社团称号。2015年元旦，和启明星工作室合作举办了元旦杯新生编程竞赛，荣获十大校级优秀社团活动称号。2015年，协会与学院合作成立的程序设计小组成员陈加伟、李雄、陈俊、熊天开、刘宏琨等人，获蓝桥杯省奖。

2016年春，协会承办了“第七届蓝桥杯全国软件和信息技术专业人才大赛”省赛，协会与学院合作成立的程序设计小组成员汪龙志、张虎、熊天开、周博、王艺、饶舸璇等人，获蓝桥杯省奖。

2017年度裴少乾荣获全国大学生数学建模比赛赛区二等奖、国际数学建模认证杯三等奖。周佩、董勇、许先志、吴佳豪、陈玮华、李均浩、何碧琳、肖三三、陈冬等15人在互联网+创新创业大赛中取得优异成绩，其中5人获得二等奖、7人获得三等奖、5人获得优胜奖。在三峡大学程序设计竞赛中李均浩获得三等奖。

2017-2018年度，王蕊在中国知网发表论文并获得三峡大学大学生跆拳道团体赛二等奖、个人赛三等奖，董勇和李文获得计算机软件著作权，汤明萩和孙建强分别获得蓝桥杯编程竞赛国家二等奖和国家三等奖，庞玮获得蓝桥杯编程竞赛省级一等奖，协会有17人获得奖学金，其中有2人获得国家励志奖学金，协会还获得“扬帆索源”悦读文化节优秀社团称号。

社团大事记

2014年，协会秉承服务与提升的发展理念，重点扶持编程部的成长，并随后与计算机与信息学院启明星工作室合作，成立程序设计小组。

2015年，为了更好的提升本校学生的计算机水平，协会进行了重大改革，协会不仅成立的创新创业部，还将原科技部细化为四个兴趣小组，即美工组、办公组、硬件组和建站组，让计算机爱好者在计协能够更细致更系统的学习到计算机知识。同时与图书馆展开合作，每周六于图书馆二楼B区203室举行计算机知识讲座与培训。

2015、2016两年，协会向学院申请到了可以长期使用的机房（7-15周周一到周日19：00-21：30），并安排协会人员值班签到，以便协会会员在机房没有安排培训的时候自学计算机知识。

2017、2018，协会与图书馆合作举办了书签设计比赛，同学们都积极报名，在校内取得良好的反响。

近十七年来，协会组织了440余次讲座，90多个培训班，二十多次全校性的大赛及其他形式多样的活动。其中有面向入门者的基础技术培训班，帮助提高水平的中级辅导班，还有追踪新技术发展的专题系列讲座，全校10000余人次参加了协会的各种活动。近年来，协会以其丰富多彩的活动、显著的工作成绩，得到了校内外的广泛好评，成为三峡大学最具影响力的社团之一。
